# README

- Ruby Version: '3.0.3'
- Rails Version: '6.1.7'
- postgreSQL Version: '1.1'


Setup:

1. git clone the repository
2. bundle install
3. rails db:create
4. rails db:migrate
5. rails server to bootup localhost:3000

Tests:

I have used minitest to run intigration and unit tests on the application
1. rails test to run the tests
